# Learning TypeScript

This repository was developed for academic and study purposes, focused exclusively on learning and practicing TypeScript. It contains multiple modules that explore the language's core features, object-oriented programming concepts, and advanced techniques applied to web development.

The project demonstrates a practical application handling "Negociações" (Transactions), covering topics such as:

- **Type Safety & Interfaces**: Utilizing TypeScript's static typing and defining contracts with Interfaces (e.g., `Comparavel`, `Modelo`)[cite: 4].
- **Object-Oriented Programming (OOP)**: Structuring applications with Classes, Inheritance, and access modifiers[cite: 4].
- **Decorators**: Implementing custom decorators for DOM injection (`@domInjector`), method execution time logging (`@logarTempoDeExecucao`), and preventing script injection (`@escape`)[cite: 4].
- **API Integration**: Fetching data from a local Node.js/Express API simulating daily transactions[cite: 4].
- **View Updates**: Dynamically updating the DOM safely and efficiently using custom View classes[cite: 4].

## Getting Started

1. **Clone the repository**: `git clone git@github.com:JVGirardi/learning-typescript.git`
2. **Navigate to the project directory**: `cd learning-typescript`

**Running the Local API:**
3. **Navigate to the API folder** (choose the appropriate module, e.g., modulo 3): `cd typescript-curso-modulo-3/servidor-api`[cite: 4]
4. **Install API Dependencies**: `npm install`
5. **Start the API**: `npm start` (The API usually runs on port 8080)

**Running the Frontend Application:**
6. **Open a new terminal and navigate to the app folder**: `cd typescript-curso-modulo-3`[cite: 4]
7. **Install App Dependencies**: `npm install`
8. **Compile TypeScript and Start**: `npm start` (This will watch for TS changes and typically start a local server like *lite-server*).
